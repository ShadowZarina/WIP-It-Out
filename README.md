# WIP-It-Out
An illustration tutorial website designed for beginners and more advanced learners! Step into the world of sketches and mixed media as we explore shapes, colors, compositions and all the concepts that build the foundation of art today! Created using HTML and CSS.

# Features
- Tutorials on three different art fundamentals, including:
  - Lines, Shapes and Forms
  - Values and Shading
  - Color Theory
- Engaging video tutorials, key terms and descriptive lessons!
- Interactive quizzes!

# What We Learned

## HTML
- Semantic tags
- Details (dropdown features)
- Implementation of video and image files
- Navigation, headers and footers

## CSS
- Flexbox
- Creating and implementing custom logos
- Media queries
